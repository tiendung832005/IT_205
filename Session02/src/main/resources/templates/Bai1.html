JPA (Java Persistence API) là một đặc tả (specification) trong Java EE dùng để quản lý dữ liệu giữa ứng dụng Java và cơ sở dữ liệu quan hệ. Nó định nghĩa một tập hợp các quy tắc và API chuẩn để thực hiện các thao tác CRUD (Create, Read, Update, Delete) mà không cần viết nhiều câu lệnh SQL thuần. Hibernate là một trong những implementation phổ biến nhất của JPA, nghĩa là Hibernate hiện thực hóa các quy tắc JPA và cung cấp thêm nhiều tính năng mở rộng mạnh mẽ. Việc sử dụng JPA giúp code trở nên dễ bảo trì hơn, giảm sự phụ thuộc vào hệ quản trị cơ sở dữ liệu cụ thể, và tách biệt rõ ràng giữa tầng dữ liệu và logic nghiệp vụ. Trong phát triển ứng dụng Java hiện đại, đặc biệt là với Spring Boot, JPA đóng vai trò cốt lõi trong việc quản lý dữ liệu hiệu quả và tối ưu hóa quá trình phát triển phần mềm.
