| **Tiêu chí** | **Spring MVC** | **Spring Boot** | | ------------------------ | ------------------------------------------------------------------------ | ----------------------------------------------------------------------------------- | | **Khái niệm** | Là một phần trong Spring Framework, hỗ trợ phát triển ứng dụng web MVC | Là một extension của Spring giúp xây dựng ứng dụng nhanh chóng và cấu hình tự động | | **Cấu hình** | Phải cấu hình thủ công: XML, file JavaConfig, servlet, view resolver,... | Cấu hình tự động (auto-configuration), rất ít cần cấu hình thủ công | | **Khởi tạo dự án** | Phải tạo cấu trúc thư mục và cấu hình từ đầu hoặc dùng Spring Tool Suite | Dùng Spring Initializr, vài click là có sẵn cấu trúc dự án mẫu | | **Quản lý phụ thuộc** | Người dùng phải chọn và thêm từng dependency vào `pom.xml` | Spring Boot starter tự gom sẵn các dependency cần thiết, dễ quản lý | | **Thời gian phát triển** | Mất thời gian set up ban đầu, dễ bị lỗi do thiếu cấu hình | Rút ngắn đáng kể thời gian phát triển, tập trung vào nghiệp vụ | | **Tính mở rộng** | Mạnh mẽ nhưng cần thêm cấu hình khi tích hợp thư viện bên ngoài | Linh hoạt và dễ mở rộng do hỗ trợ cấu hình theo chuẩn convention-over-configuration | | **Phù hợp với ai?** | Dành cho những người cần tùy biến chi tiết cấu hình hoặc dự án đặc thù | Dành cho dev muốn build nhanh, dễ maintain, nhất là microservices |
